# Supremo — soysupremohn.com

Sitio oficial premium de Supremo.

## Stack

- Next.js 16 + React 19 + TypeScript
- Tailwind CSS 4 + Motion
- Supabase (producción) / `data/store.json` (local)
- Resend (contacto)
- Vercel + GitHub Actions sync

## Desarrollo

```bash
npm install
cp .env.example .env.local
npm run dev
```

Admin local por defecto: `admin` / `admin` (cámbialo en `.env.local`).

## Scripts

- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run typecheck`
